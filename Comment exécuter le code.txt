Tout d'abord, vous devez ouvrir le répertoire Code_python et vous assurer 
que votre chemin d'exécution est bien TP_Data_Augmentation\TP_Data_Augmentation\Code_python>. 
Par la suite, vous devez vous assurer d'installer toutes les bibliothèques nécessaires qui 
se trouvent dans le fichier requirements.txt en exécutant cette commande :
pip install -r requirements.txt

Assurez-vous d'être dans le bon répertoire pour que la commande puisse
trouver le fichier requirements.txt et installer correctement les dépendances.
